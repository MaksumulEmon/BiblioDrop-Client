# 📖 Project Overview

BiblioDrop is a modern online book delivery management system designed to simplify the process of publishing, browsing, requesting, and delivering books.

The platform provides three different dashboards for Readers, Librarians, and Administrators, each with dedicated permissions and management tools.

Readers can discover books, place delivery requests, leave reviews, and manage their reading history.

Librarians can publish books, manage inventory, process delivery requests, and monitor their own books.

Administrators have complete control over the platform, including book approval, user management, transaction monitoring, and system-wide moderation.

The application is fully responsive, secure, and optimized for modern web browsers.
